TAG : !DOCTYPE

!DOCTYPE A declaração DOCTYPE é usada em HTML para especificar o tipo de documento e a versão do HTML que está sendo utilizada na página da web. Ela é colocada no início do documento HTML, antes da tag HTML, para garantir que o navegador renderize a página corretamente e siga as regras apropriadas para analisar o conteúdo.

Em português, a seção "head" de um documento HTML é chamada de "cabeçalho". O cabeçalho é uma parte fundamental de um documento HTML e contém informações importantes sobre a página, como metadados, títulos, links para folhas de estilo, scripts e outras informações que não são diretamente visíveis na página, mas são cruciais para o funcionamento e a aparência correta do documento. title: Título da Página: Define o título da página, que aparece na barra de título do navegador e pode ser usado por mecanismos de busca.

TAG : BODY

A tag body é outra parte fundamental de um documento HTML. Ela é usada para definir o conteúdo visível da página, ou seja, tudo o que é exibido no navegador e faz parte da interface que os usuários interagem.
Aqui estão algumas das principais funções da tag body:
Conteúdo da Página: A tag body é o local onde todo o conteúdo visível da página é colocado. Isso inclui textos, imagens, vídeos, formulários, links e outros elementos HTML que os usuários podem ver e interagir.
Estrutura da Página: O conteúdo dentro da tag body define a estrutura da página. Isso inclui a organização dos elementos, como cabeçalhos, parágrafos, listas, tabelas e divisões (tags div), que são utilizados para criar a estrutura visual da página.
Inclusão de Mídia: A tag body é o local onde você pode incluir elementos de mídia, como imagens (tag img), áudio (tag audio), vídeo (tag video) e outros recursos multimídia que serão exibidos aos usuários.
Links e Navegação: Os links (tag a) são comumente usados dentro da tag body para criar navegação entre páginas ou para direcionar os usuários para outras partes do conteúdo da página.
Formulários: Caso sua página contenha formulários (tag form), eles devem estar dentro da tag body. Formulários são usados para coletar informações dos usuários e enviá-las ao servidor.

TAG : SPAN

A tag span é um elemento em HTML usado para estilizar um trecho específico de texto ou para agrupar elementos em linha para aplicar algum estilo ou comportamento específico.
Diferente da tag div, que é usada para agrupar blocos de conteúdo em linha, a tag span é usada para agrupar elementos em linha sem causar quebras de linha ou mudanças de layout. Exemplo de uso:



Neste exemplo, usamos a tag span para envolver duas partes diferentes do texto. A primeira parte está sendo estilizada com a propriedade CSS "color: blue;" para tornar o texto azul, enquanto a segunda parte é estilizada com "font-weight: bold;" para torná-la em negrito. Esses estilos serão aplicados apenas às partes específicas dentro do span e não afetarão o resto do parágrafo.
A tag span é frequentemente usada em conjunto com folhas de estilo CSS ou com atributos "style" para aplicar estilos específicos a partes do texto ou para adicionar interatividade por meio de JavaScript. Ela é uma tag flexível e útil quando é necessário aplicar estilos ou funcionalidades a pequenos trechos de texto ou elementos em linha.

A tag select é amplamente usada em formulários e é útil quando há um conjunto de opções predefinidas e o usuário precisa selecionar uma delas. Ela permite que os usuários façam escolhas de forma mais intuitiva e organizada.

A tag textarea é muito útil quando você precisa coletar informações mais extensas dos usuários e oferece uma forma conveniente de permitir que eles insiram texto de várias linhas de maneira intuitiva.
